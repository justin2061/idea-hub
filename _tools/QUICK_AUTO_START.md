## 🚀 步驟 1：設置 API Key（2 分鐘）

### 1.1 獲取 Anthropic API Key

1. 訪問：https://console.anthropic.com/
2. 註冊/登入
3. 前往 API Keys 頁面
4. 點擊 "Create Key"
5. 複製 Key（格式：`sk-ant-api03-...`）

### 1.2 添加到 GitHub Secrets

1. 進入你的 Repository
2. 點擊 `Settings`
3. 左側菜單選擇 `Secrets and variables` → `Actions`
4. 點擊 `New repository secret`
5. 填寫：
   - Name: `ANTHROPIC_API_KEY`
   - Secret: 貼上你的 API Key
6. 點擊 `Add secret`

✅ **完成！**

---

## 🚀 步驟 2：啟用 GitHub Actions（1 分鐘）

### 2.1 確認 Actions 已啟用

1. Repository → `Settings`
2. 左側選擇 `Actions` → `General`
3. 確保選擇 "Allow all actions and reusable workflows"
4. 點擊 `Save`

### 2.2 確認工作流文件已存在

```bash
# 檢查文件是否存在
ls -la .github/workflows/fully-auto-content.yml
```

如果不存在，請先提交工作流文件（應該已經在 repository 中了）。

✅ **完成！**

---

## 🚀 步驟 3：手動觸發第一次運行（2 分鐘）

### 3.1 觸發工作流

1. 進入 Repository
2. 點擊 `Actions` 標籤
3. 左側選擇 `🤖 Fully Auto Generate & Publish`
4. 點擊右側 `Run workflow` 按鈕
5. 設置參數：
   ```
   Categories: all
   Count: 1
   Skip validation: 不勾選
   ```
6. 點擊綠色的 `Run workflow` 按鈕

### 3.2 等待運行完成（5-10 分鐘）

系統會：
1. 🔍 搜尋 6 個分類的熱門話題
2. 🤖 使用 AI 生成 6 篇文章
3. ✅ 驗證文章品質
4. 📝 提交到 GitHub
5. 🚀 自動部署到網站
6. 📊 創建報告 Issue

### 3.3 查看結果

**查看運行狀態：**
- Actions 頁面會顯示運行進度
- 綠色勾號 = 成功
- 紅色叉號 = 失敗（查看日誌）

**查看生成的文章：**
```bash
# 查看今天生成的文章
ls -la _posts/$(date +%Y-%m-%d)-*.md
```

**查看報告 Issue：**
- Repository → Issues
- 找到最新的 `📊 自動生成報告` Issue

**訪問網站：**
- https://your-username.github.io/idea-hub
- 等待 2-5 分鐘後文章會出現

✅ **完成！**

## ⚙️ 可選設置

### 設置 Slack 通知（可選）

如果你想收到 Slack 通知：

1. 創建 Slack Incoming Webhook
2. 添加到 GitHub Secrets：
   - Name: `SLACK_WEBHOOK`
   - Secret: 你的 webhook URL

### 調整運行時間

編輯 `.github/workflows/fully-auto-content.yml`：

```yaml
schedule:
  # 每天早上 9:00
  - cron: '0 1 * * *'

  # 或每週一 9:00
  - cron: '0 1 * * 1'

  # 或保持默認（週一、三、五）
  - cron: '0 1 * * 1,3,5'
```

### 調整生成數量

在工作流文件中修改 `default` 值：

```yaml
count:
  default: '2'  # 改成每分類 2 篇
```

## 🐛 遇到問題？

### 常見問題

**Q: 工作流沒有運行？**

A:
1. 檢查 GitHub Actions 是否啟用
2. 手動觸發一次測試
3. 查看 Actions 日誌

**Q: 生成失敗？**

A:
1. 檢查 API Key 是否正確
2. 檢查 API 額度是否用完
3. 查看詳細錯誤日誌

**Q: 文章質量不好？**

A:
1. 提高 MIN_WORDS 設置
2. 調整 prompt 模板
3. 人工審核並優化

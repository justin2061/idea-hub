### ✨ 新工作流程的優勢

1. **無縫寫作體驗**
   - 在 Obsidian 中專注寫作
   - 自動 Git 提交和推送
   - 5 分鐘內自動部署到網站

2. **強大的編輯功能**
   - 即時預覽
   - 雙向連結
   - 圖表視覺化
   - 標籤和分類管理

3. **版本控制自動化**
   - 每次儲存自動建立 Git 提交
   - 完整的版本歷史記錄
   - 雲端備份保護

### 🚀 如何開始使用

```bash

# 2. 開啟 content/posts/ 作為 Vault

# 4. 開始寫作！
```
